## サービス概要
  日焼け止めの塗り直しをリマインドするサービスです。

  その日の紫外線の強さによって適切な日焼け止めの強さを提案、あなたの予定に合わせてLINEで日焼け止めの塗り直しをLINEでリマインドします。

## メインのターゲットユーザー
  日焼け止めを毎日丁寧に塗っても朝しか塗っていない人
  日焼け止めを夏の期間しか塗らない人
  そもそも日焼け止めを塗る習慣がない人
